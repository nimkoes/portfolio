# Agent Definition Template

## 역할

이 agent가 맡는 책임을 한 문장으로 적습니다.

## 주요 입력

- 읽어야 할 파일
- 필요한 실행 결과
- 필요한 사용자 정보

## 해야 할 일

1. 핵심 작업을 적습니다.
2. 판단 기준을 적습니다.
3. 완료 기준을 적습니다.

## 하지 말아야 할 일

1. 넘지 말아야 할 책임 범위를 적습니다.
2. 위험한 행동을 적습니다.

## 출력 형식

```json
{
  "summary": "작업 요약",
  "changedFiles": [],
  "verification": [],
  "risks": []
}
```
